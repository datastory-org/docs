## About Datastory and the API

Datastory helps organizations define build websites and visualization from different datasets. Using the API you can interact with the core aspects of the platform. Here are some examples of what the API makes possible:

* Create Models and Fetch Items

* Organize Statistics into Data Cubes&#x20;

* Upload CSV files

* Create Visualizations

* Retrieve Insights from your Data
